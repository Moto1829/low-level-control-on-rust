# はじめに

## このコンテンツについて

RustはC/C++と同等の実行速度を持ちながら、メモリ安全性を保証するシステムプログラミング言語です。
しかし「安全に書ける」だけでなく、**意図的に低レベル制御を行うことでさらなる高速化が可能**です。

このコンテンツでは以下を目指します。

- Rustの抽象化の「下」で何が起きているかを理解する
- `unsafe` を適切に使い、パフォーマンスを最大化する手法を学ぶ
- OSのシステムコールやCPU命令を直接扱う実践力を身につける

## 前提知識

- Rustの所有権・借用・ライフタイムを理解していること
- `cargo` による基本的なプロジェクト管理ができること
- C言語の基本的な知識があるとFFIの章で役立ちます

## 開発環境

```bash
# Rustツールチェインのインストール（まだの場合）
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# nightly（SIMD・asm章で使用）
rustup toolchain install nightly

# mdBookのインストール
cargo install mdbook
```

## 注意事項

`unsafe` コードは正しく使えば強力なツールですが、誤用するとメモリ破壊・未定義動作の原因になります。
各章では「なぜunsafeが必要か」「どこが危険か」を明確にしながら解説します。
